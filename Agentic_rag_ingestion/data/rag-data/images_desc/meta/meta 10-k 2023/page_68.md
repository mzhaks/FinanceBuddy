**Text Summary:**
Key Facebook metrics include Daily Active Users (DAUs), Monthly Active Users (MAUs), and Average Revenue Per User (ARPU). These metrics typically exclude Instagram, WhatsApp, or other products unless specified. User numbers impact revenue and financial results. Daily Active Users (DAUs) are defined as a user who visited Facebook or used Messenger (while logged into Facebook) on a given day, measuring user engagement. For geographic reporting, Europe includes users in Russia and Turkey, while Rest of World includes users in Africa, Latin America, and the Middle East.
